# Documentation

This folder contains supporting documents for the COMP3000 project.

Expected files include:

- `project_report.pdf`
- `poster.jpg`
- `ai_declaration.pdf`
- `viva_slides.pdf` or `viva_slides.pptx`

These files support the final ePortfolio submission and provide evidence of the project work, including planning, implementation, evaluation, ethics, and reflection.

The report explains the full project in detail, while the poster and viva materials summarise the key points for presentation and discussion.
