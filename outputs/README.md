# Outputs

This folder contains the main results produced during the project.

It is used to store the files that show how the model performed and how the results were analysed.

Expected files include:

- `confusion_matrix.png`
- `roc_curve.png`
- `gradcam_examples/`

The confusion matrix shows where the model was correct and where it made mistakes. The ROC curve shows how well the model separated real and fake images. The Grad-CAM examples help show which parts of the image influenced the model’s prediction.

These outputs support the evaluation section of the project and help show that the model was tested, not only built.
